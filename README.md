# Spring Boot Sample Application
## Running this app locally
```cmd
git clone git@github.com:Casual-P/SpringBootProj.git
cd springbootproj
./mvnw package
docker-compose up -d --build
```
Then you can access to this app at http://localhost/api/view/main
